Description

Data model class for a Google Apps Email List Recipient Entry.

Each instance of this class represents a recipient of an email list hosted on a Google Apps domain. Each email list may contain multiple recipients. Email lists themselves are described by Zend_Gdata_EmailListEntry. Multiple recipient entries are contained within instances of Zend_Gdata_Gapps_EmailListRecipientFeed.

To transfer email list recipients to and from the Google Apps servers, including creating new recipients, refer to the Google Apps service class, Zend_Gdata_Gapps.

This class represents in the Google Data protocol.
